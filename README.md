Discord üzerinden DnD oynarken hızlı zar atabilmek için oluşturulmuş bir python scripti. 

## Kullanımı

- ```cp roll.py /usr/bin/```
- ```roll zar```

- ```roll yardim``` yazarak hangi zarları atabileceğinizi görebilirsiniz.
